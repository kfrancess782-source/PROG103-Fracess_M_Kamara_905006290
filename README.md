# Student Attendance Tracking System (SATS)

## Overview
The Student Attendance Tracking System (SATS) is a terminal-based Python application developed for PROG103 Structured Programming.

The system allows users to:
- Record student attendance
- Mark students as Present, Absent, or Late
- Generate attendance remarks
- Display attendance summaries
- Validate user input

## Features
- Structured programming approach
- Functions and loops
- Input validation
- Attendance summary reporting
- User-friendly terminal interface

## Technologies Used
- Python 3
- Visual Studio Code
- Git & GitHub
